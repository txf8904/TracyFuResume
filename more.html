Beyond UX Design,

I'm also a game designer.

From my point of view, UX design and game design are very similar, they are like cousins to each other. UX design is more about creating a friendly user experience for the users, and game design creates a user experience with the challenge.

UX design and game design both belong to human-computer interaction, which focuses on the interaction between the users and computers. I have a great passion for this topic and would like to learn more about this topic in the future.
